
# BingoBasic
A basic bingo game you play with a friend

## Built with
- Java

## Installation
- Download files
- Go to command line
- After directing the command line to the folder
  - Enter "javac BingoGame.java" in the command line
  - Enter "java BingoGame" in the command line
